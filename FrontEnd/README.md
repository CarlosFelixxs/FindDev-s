# FindDev-s Frontend

React/TypeScript frontend for the FindDev-s platform prototype.

The app includes public pages, authentication screens, profile flows, vacancy screens and reusable UI components for a freelance developer marketplace experience.

## Stack

- React 18
- TypeScript
- React Router
- Axios
- React Hook Form
- Yup
- CSS Modules

## Scripts

```bash
npm install
npm start
npm run build
```

## API

The frontend uses Axios through `src/services/api.ts`.

The original project points to an external backend URL used during the academic project. For a production-grade setup, this should be moved to environment variables before deployment.

## Status

This frontend is part of an older full-stack academic project and is kept public as a portfolio case study.
